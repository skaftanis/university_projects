Στον φάκελλο sqlCode βρίσκονται τα ζητούμενα αρχεία sql, ενώ στον φάκελλο javaCode η Java.

Για την εκτέλεσή του, αρκεί να τρέξετε το Project.jar σε οποιοδήποτε λειτουργικό σύστημα με java. 

Το project υλοποιήθηκε στο περιβάλλον eclipse. Αν επιθυμείται για κάποιο λόγο την εκτέλεσή του από εκεί χρειάζεται να κάνετε import το project και να προσθέσετε το mysql-connector-java-5.1.36-bin.jar στα libraries του build path.

Για τη σύνδεση με τη βάση δεδομένων πρέπει να κάνετε τις κατάλληλες ρυθμίσεις από τα settings του προγράμματος. Από προεπιλογή είναι ρυθμισμένο να δουλεύει με τα στοιχεία της δικής μου βάσει, συνεπώς θα χρειαστεί να αλλάξετε τον κώδικο, ίσως και το username. Οι ρυθμίσεις δεν αποθηκεύονται για τις επόμενες συνεδρίες.
